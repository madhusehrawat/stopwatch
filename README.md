⏱ Stopwatch Web App
📌 Description

This is a simple Stopwatch web application built using HTML, CSS, and JavaScript.
The application allows users to start, stop, and reset the timer. It is useful for measuring time intervals and practicing basic JavaScript functionality.

✨ Features

Start the stopwatch

Stop / Pause the stopwatch

Reset the timer

Simple and clean user interface

🛠 Technologies Used

HTML

CSS

JavaScript
